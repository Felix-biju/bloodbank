# Blood Bank Management System

A web-based application to manage blood donors and donation records.

## Features
- **Dashboard**: Overview of total donors, donations, and eligible donors.
- **Donor Registry**: Register new donors with personal and medical details.
- **Record Donations**: Log blood donations to keep track of donor eligibility.
- **Search**: Find eligible donors quickly based on blood group and location.

## Technology Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript
- **Database**: Browser LocalStorage (No backend server required)

## How to Run
This project is fully static and runs in the browser. 
To view it, simply open the `index.html` file in any modern web browser or visit the live GitHub Pages link.
